# 2×2 像素角色動畫提示詞

把 `[角色描述]`、`[方向]`、`[動作]` 換成需要的內容，並附上角色參考圖。

## 共用提示詞

```text
請嚴格參考附件角色，製作一張正方形 2×2 像素風 Sprite Sheet。

角色：[角色描述]
視角：[方向：正面／左側面／背面]
動作：[動作：待機／走路／使用十字鎬挖礦]

四格播放順序固定為：左上 → 右上 → 左下 → 右下。
四格必須是同一名角色，臉、髮型、帽子、服裝、顏色、身材比例與像素畫風完全一致。
每格大小完全相同，角色置中，腳底對齊同一基準線，四周保留安全距離，不可跨格。
輸出 1024×1024 PNG，背景真正透明（alpha），不要畫灰白棋盤格。
不要文字、標籤、格線、邊框、介面、陰影背景或額外角色。
像素邊緣清晰，不要模糊，不要平滑插值。
```

## 動作補充

- 待機：四格只做小幅呼吸、眨眼或重心變化，不能改變站位。
- 走路：四格形成完整循環，左右腳交替，身體高度變化要小。
- 挖礦：第一格蓄力、第二格舉鎬、第三格落鎬、第四格命中並出現少量碎石；角色位置保持一致。

生成後請確認圖片沒有把透明背景畫成棋盤格，再上傳到遊戲後台。
